# react-node-typescript-postgres-starter

**Okta Setup Notes**

-   Create and confirm an [Okta](https://okta.com) developer account
-   Log into the Okta developer dashboard
-   Get your Org URL
-   Get your API Token
-   Add these values to `.env` file
